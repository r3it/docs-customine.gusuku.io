## Summary

**For use in Japanese character.**

Converts the specified field value from half-width to full-width. The conversion target is the following characters:

-	Numeric
-	Alphabetic
-	Symbol
-	Kana

## How to use

### Parameters to set

Set the Fields.

#### field

Select the fields that you want to convert.
